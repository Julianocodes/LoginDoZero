# 🔐 LoginDoZero

Projeto de sistema de login desenvolvido do zero utilizando HTML, CSS e JavaScript puro.

## 🚀 Sobre o Projeto

O objetivo deste projeto foi praticar conceitos fundamentais do desenvolvimento front-end criando um sistema completo de autenticação simples, contendo:

* Tela de Login
* Tela de Cadastro
* Recuperação de Senha
* Validação de Formulários
* Navegação entre páginas
* Manipulação do DOM
* Validação de Email e Senha com JavaScript

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## 📚 Conceitos Praticados

Durante o desenvolvimento deste projeto foram estudados e aplicados:

* Eventos com JavaScript
* addEventListener()
* Condições (`if`)
* Validação de formulários
* Manipulação de inputs
* `.trim()`
* `.includes()`
* `.checked`
* Organização de código
* Estrutura de páginas web
* Responsividade básica

---

## 💻 Funcionalidades

✅ Cadastro de usuário
✅ Validação de email
✅ Validação de senha
✅ Confirmação de senha
✅ Checkbox de termos
✅ Login com validação
✅ Página de recuperação de senha

---

Esse projeto faz parte da minha jornada aprendendo programação e desenvolvimento web do zero, praticando lógica, estruturação de páginas e JavaScript na prática.
